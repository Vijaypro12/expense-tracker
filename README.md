# Expense Tracker & Financial Report Analyzer

A Python-based expense tracking application that helps users record, analyze, and visualize daily expenses using CSV as a lightweight database.

## Features
- Add and store daily expenses (date, amount, category, description)
- CSV-based data storage (no external DB required)
- Expense analysis using Pandas
- Category-wise and date-wise expense aggregation
- Visual reports using Matplotlib

## Tech Stack
- Python
- Pandas
- Matplotlib
- CSV
